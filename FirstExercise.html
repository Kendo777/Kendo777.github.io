Dynamic Websites:

1. What is a front-end developer? And a back-end developer? And full-stack developer?

A Frontend Developers are responsible for layout the semantic structure of content (HTML), encoding the layout into style sheets (CSS), and adding user interaction (JavaScript).

A Backend Developer is a server-side programmer. Allowing everything we see when interacting with an app or website to work.

A full stack web developer is a person who can develop both client and server software. In addition to mastering HTML and CSS, he/she also knows how to: Program a browser (like using JavaScript, jQuery, Angular, or Vue) Program a server (like using PHP, ASP, Python, or Node).

3. What does HTML stand for? What is its role in dynamic webpages?

HTML stands for Hypertext Markup Language. It allows the user to create and structure sections, paragraphs, headings, links, and blockquotes for web pages and applications. (not a programming language)

Classical hypertext navigation, with HTML or XHTML alone, provides "static" content, meaning that the user requests a web page and simply views the page and the information on that page.

6. What does CSS stand for? What is its role in dynamic web pages?

CSS (cascading style sheets) is a language that defines the appearance of a document written in a markup language (for example, HTML). Web page elements created with HTML will be given the desired appearance using CSS: colours, spaces between elements, fonts, ... thus separating the structure of the presentation.

8. What is a DNS? How does it work?

The Domain Name System (DNS) is a hierarchical and decentralized naming system for computers, services, or other resources connected to the Internet or a private network.

It associates various information with domain names assigned to each of the participating entities. Most prominently, it translates more readily memorized domain names to the numerical IP addresses needed for locating and identifying computer services and devices with the underlying network protocols. By providing a worldwide, distributed directory service, the Domain Name System has been an essential component of the functionality of the Internet since 1985.
